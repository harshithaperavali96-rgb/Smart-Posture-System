# Smart Posture System

## Overview
The Smart Posture System is an AI-powered application that monitors a person's sitting posture using a webcam. It uses MediaPipe Pose and OpenCV to detect body landmarks in real time.

## Features
- Real-time posture detection
- Webcam-based monitoring
- Pose landmark detection
- Easy to use
- Beginner-friendly project

## Technologies Used
- Python
- OpenCV
- MediaPipe
- NumPy

## Installation

```bash
pip install -r requirements.txt
```

## Run

```bash
python posture_detector.py
```
